## ✨ Features

| Category | Features |
|----------|----------|
| **🔑 License Management** | Generate single/bulk licenses, Lifetime/Time-limited keys, Automatic key generation |
| **👥 Customer System** | Customer profiles, License assignment, Check-in tracking |
| **📦 Product Management** | Multiple products, Product-based licensing, Price tracking |
| **🛡️ Security** | IP Blacklisting (with wildcard support), Session-based admin auth, Request logging |
| **📊 Dashboard** | Real-time statistics, Active/Lifetime license counters, Expiring soon alerts |
| **🤖 Discord Integration** | Webhook notifications, Bot commands (!check, !customer), Real-time alerts |
| **🌐 API** | RESTful endpoints, JSON responses, Customer license lookup |

## 🚀 Quick Start

## Installation
1. Clone or download the project
git clone https://github.com/S1rChips/Vanily-License-System.git
cd vanily-license-manager

2. Install dependencies
pip install flask requests

3. (Optional) For Discord bot integration
pip install discord.py

4. Run the application
python app.py
